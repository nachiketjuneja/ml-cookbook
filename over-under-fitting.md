OVERFITTING AND UNDERFITTING

LEr's say we only had 2 data points. Now you created line of best fit.
Tomorrow we get more data points apart from training data, there is a possibility there would be difference between predicted points and actual points. This means there's overfitting.

OVERFITTING
Model trained well with training data and performs well, but fails to perform with test data (Real data)
This condition is called **Low bias** for training data.
This condition is called **high variance** for test data.

Whenever we talk about bias - training data
Whenever we talk about variance - test data

UNDERFITTING
Model fails to perform with training data and fails with test data.
Model accuracy bad with test as well as training data.

Overfitting - Low bias, high variance
Underfitting - High bias, high variance
